# Karan Portfolio

Personal portfolio website: Analytics, Strategy & AI Solutions.

Built from a static Google Sites page into a custom coded, animated site.

## Live site
https://karan-portfolio-project.vercel.app/

## How it was built
- Claude Code in VS Code to vibecode the site
- Claude for structural and design direction
- ChatGPT for text writeups and mock image generation

## Tech stack
- HTML, CSS, JS (no framework, no build step)
- GSAP for scroll animations
- Simple Icons + local assets for logos
- Google Fonts

## Structure
Five static pages: index, about, experience, projects, ai-lab.
